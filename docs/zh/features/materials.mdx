## 支持的文件格式

| 格式 | 说明 |
|------|------|
| PDF | 提取文本和嵌入图片 |
| DOCX / DOC | 解析文档内容和结构 |
| PPTX / PPT | 提取幻灯片内容 |
| XLSX / XLS / CSV | 解析表格数据 |
| Markdown | 读取格式化文本和图片链接 |
| TXT | 纯文本内容 |

## 素材工作流程

1. 在首页或详细编辑器中上传文件
2. 后端自动解析提取关键内容、图片和图表信息
3. 提取的内容作为上下文用于生成幻灯片描述和图片

## 图片粘贴

可直接在首页、大纲编辑器或描述卡片中粘贴图片，系统会立即识别并整合。

## 风格参考

上传参考图片或模板引导整个演示文稿的视觉风格，AI 会匹配参考的配色、布局和设计语言。
