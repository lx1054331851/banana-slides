description: "从想法、大纲或页面描述开始"
---

## 从想法开始

最简单的方式。在首页输入主题，Banana Slides 会：

1. 生成带章节的结构化大纲
2. 为每页创建详细描述
3. 根据描述生成幻灯片图片

## 从大纲开始

如果已有结构思路：

1. 创建项目，进入大纲编辑器
2. 添加、删除或重排页面和章节
3. 用自然语言调整大纲（如"加一节市场分析"）
4. 从大纲生成页面描述

## 从页面描述开始

需要最大控制力时，在详细编辑器中直接编写每页描述，告诉 AI 渲染什么内容、布局和风格。

<Tip>
  在描述中包含具体文字内容，可显著提升生成幻灯片的文字渲染质量。
</Tip>

## 风格定制

- **参考图片** — 上传风格参考图引导视觉风格
- **模板** — 使用文字预设控制外观
- **逐页素材** — 为单页附加特定图片或图表
