## 格式

| 格式 | 可编辑 | 说明 |
|------|--------|------|
| PPTX | 是 | 文字、图片和布局可在 PowerPoint 中编辑 |
| PDF | 否 | 像素级精确输出，可直接演示 |

两种格式默认 16:9 比例。

## 可编辑 PPTX（Beta）

可编辑导出会从生成的幻灯片图片中提取文字，重建为原生 PowerPoint 文本元素，保留：

- 字号、颜色和加粗样式
- 文字定位
- 表格内容

<Note>
  配置 `BAIDU_API_KEY` 可获得最佳效果，详见[配置说明](/zh/configuration#百度-api-key)。
</Note>

## 页面选择

可选择特定页面导出，无需导出整个演示文稿。在幻灯片预览中使用多选功能选择要包含的页面。
